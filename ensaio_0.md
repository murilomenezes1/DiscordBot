Para criar o bot, segui o passo-a-passo fornecido pelo chatGPT, que se encontra no readme da APS no repositorio da disciplina.
Utilizei o ChatGPT como fonte, evitando longas pesquisas, para implementar a feature em que o bot responde apenas mensagens privadas.
Por fim, clonei este repositorio na VM do monstrao, instalei as bibliotecas necessarias e executei o arquivo .py usando o nohup. Foi necessario, tambem, criar o arquivo .env na VM, que contem o token do bot.
Uma dificuldade encontrada foi a ausencia da VPN, sendo necessario estar presencialmente no insper para executar o bot na VM do monstrao.
